== Description ==

Adds Twitter Anywhere to your wordpress blog

Requirement:
You should have Twitter API Key


== Installation ==

1. Unzip WPTwitterConnect.0.1.zip
2. Upload `wp-twitter-connect` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure the settings using "WP Twitter Connect" under Settings panel.
